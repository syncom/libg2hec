3.1 A web demo of divisor scalar multiplication

One used to be able to play around with divisor scalar multiplication on the following demo page (thanks to Purdue’s Mathematics Department):

http://www.math.purdue.edu/~nshang/g2hec/demo.html.

The demo page showed an application built on top of the G2HEC library. Although you may smell a bit of mathematics from this demo, in general programmers can build cryptographic algorithms/protocols without knowing any college-level math.


The hosting is no longer available as of 2021. Interested readers are encouraged to explore the [demo/](https://github.com/syncom/libg2hec/tree/master/demo) directory in the source repository.
